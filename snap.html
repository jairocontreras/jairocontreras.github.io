Snap

Enhance keyboard shortcuts.

FEATURES

  • Hold Shift to extend window
  • Preserve window resize

REQUIREMENTS

Install AutoHotkey

NOTES

These system settings will be disabled in effect:

  • When I snap a window, show what I can snap next to it
  • When I resize a snapped window, simultaneously resize any adjacent snapped window

Run as administrator in order to control administrator-level windows
