Inbox for Gmail

Get notified of incoming mail in the system tray.

FEATURES

  • Empty Unread *hover icons for demo
  • Supports multiple accounts, individual icons, and different colors

REQUIREMENTS

  • Google account security setting: Allow less secure apps
  • Passwords cannot contain whitespace
  • Install AutoHotkey

INSTALLATION

Run install.ahk

SETUP

  • How to add account(s)
    1. Open accounts.txt
    2. Enter login credentials
    3. Choose color from icons folder
    4. Add one account per line
  • How to start app: open launch.ahk
  • How to add to system startup
