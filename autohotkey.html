Notes

  • Run as administrator
  • Add to startup using Task Scheduler
    1. Create Task
    2. General
      • Name: "script.ahk"
      • Run with highest privileges
    3. Triggers
      • Begin the task: At log on
      • Delay task for: 1 second (for system tray icon to show)
    4. Actions
      • Program/script: "path_to_AutoHotkey.exe"
      • Add arguments (optional): "path_to_script.ahk"

    Or run the following command as administrator:

    schtasks /create /tn "script.ahk" /tr "\"path_to_AutoHotkey.exe\" \"path_to_script.ahk\"" /sc onlogon /delay 0000:01 /rl highest
