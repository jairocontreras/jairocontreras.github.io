Inbox for Gmail™

Get notified of incoming mail in the system tray.

EXAMPLE

FEATURES

  • Add multiple accounts
  • Open-source; fully customizable

REQUIREMENTS

  • Google account security setting: Allow less secure apps
  • Passwords cannot contain whitespace
  • Install AutoHotkey

INSTALLATION

Run install.ahk

SETUP

  • How to add to system startup
  • How often does it check mail: every minute
  • How to add account: open accounts.txt
