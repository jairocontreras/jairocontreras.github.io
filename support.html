Troubleshooting Steps

If the extension is not working:

Part 1

  • Go to Manage extension
  • Enable Developer mode
  • Enable Collect errors
  • Try extension
  • Go to chrome://extensions
  • Check for Errors (next to Details and Remove)
  • Copy and send error messages, if any

Part 2

  • Close open browser windows
  • Run chrome --enable-logging
  • Try extension
  • Switch to debugging window
  • Check for ERROR messages
  • Copy and send, if any

Alternatively, support is available through remote access software
